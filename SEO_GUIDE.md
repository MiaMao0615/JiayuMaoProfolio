## 🚀 部署后的 SEO 步骤

### 步骤 1: 部署到 GitHub Pages

按照 `QUICK_START.md` 的步骤部署网站后，你会得到一个网址：
```
https://MiaMao0615.github.io/你的仓库名/
```

### 步骤 2: 提交到 Google 搜索引擎

1. **Google Search Console**
   - 访问：https://search.google.com/search-console
   - 点击 "添加属性"
   - **选择 "URL prefix"（URL 前缀）** ← 选择这个！
   - 输入你的网站 URL（如：`https://MiaMao0615.github.io/portfolio/`）
   - 点击 "继续"（CONTINUE）
   - 选择验证方法（推荐选择 "HTML 文件" 或 "HTML 标记"）

2. **提交网站地图**
   - 在 Google Search Console 中，点击左侧 "网站地图"
   - 输入：`sitemap.xml`
   - 点击 "提交"
   - **注意**：部署后需要更新 `sitemap.xml` 中的 URL 为你的实际网址

### 步骤 3: 提交到其他搜索引擎

- **Bing Webmaster Tools**: https://www.bing.com/webmasters
- **百度站长平台**（如果主要面向中文用户）: https://ziyuan.baidu.com/

### 步骤 4: 更新 sitemap.xml

部署后，编辑 `sitemap.xml` 文件，将 URL 替换为你的实际网址：

```xml
<loc>https://MiaMao0615.github.io/你的仓库名/</loc>
```

然后重新提交到 GitHub。
